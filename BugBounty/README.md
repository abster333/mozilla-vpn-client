# Mozilla VPN Client - Bug Bounty Resources

This folder contains resources and information to assist in bug bounty research for the Mozilla VPN Client.

## Contents

*   **[Program Information](BOUNTY_INFO.md)**: Scope, rewards, and reporting guidelines.
*   **[Project Structure](PROJECT_STRUCTURE.md)**: Overview of the codebase architecture and key directories.
*   **[Historical Vulnerabilities](HISTORICAL_VULNS.md)**: Analysis of past security fixes and recurring bug themes.
*   **[Hunting Tips](HUNTING_TIPS.md)**: Strategies, attack surfaces, and tools for finding vulnerabilities.

## Quick Start

1.  **Build the project**: Follow the instructions in `../docs/Building/macos.md` (or your platform).
2.  **Read the Architecture**: Understand the Client-Daemon split in `PROJECT_STRUCTURE.md`.
3.  **Check the Tips**: Look at `HUNTING_TIPS.md` for high-value targets like IPC.
4.  **Happy Hunting!**
